# 🏠 House Price Prediction using Machine Learning

A complete Machine Learning project that predicts house prices based on various features using **XGBoost Regression**.

---

## 🚀 Project Overview

This project focuses on building a regression model to accurately predict house prices. It follows a standard ML pipeline including data preprocessing, analysis, model training, and evaluation.

---

## 🧠 Model Used

- 🔹 XGBoost Regressor (Extreme Gradient Boosting)
- Highly efficient and powerful for structured/tabular data

---

## 📊 Workflow

The project follows these steps:

1. 📥 **House Price Data**
   - Dataset loading and understanding

2. 🧹 **Data Preprocessing**
   - Handling missing values  
   - Encoding categorical variables  
   - Feature scaling (if required)

3. 📈 **Data Analysis**
   - Correlation analysis  
   - Visualization of important features  

4. 🔀 **Train-Test Split**
   - Splitting dataset into training and testing sets  

5. 🤖 **Model Training**
   - Training using XGBoost Regressor  

6. 📏 **Model Evaluation**
   - Metrics used:
     - Mean Absolute Error (MAE)
     - Mean Squared Error (MSE)
     - R² Score
